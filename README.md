## Api key
There is an option to use an Github api for if you run out of pull requests

1. Go to <a href=https://github.com/settings/personal-access-tokens>github.com</a>
2. Go to generate new token
3. Give your token an name 
4. Set a expiration date of your choosing
5. Go down and press on generate token
6. Press the api button on the website and enter your api key

now your good to go and use it as much as you want 

ps: Normally it is 60 requests an hour but with api it is 5000 requests an hour.
